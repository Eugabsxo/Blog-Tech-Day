# Blog-Tech-Day
O site simula um blog de notícias sobre tecnologia, inteligência artificial e metaverso, com seções bem organizadas, visual limpo e navegação clara. A proposta é mostrar domínio das principais ferramentas de front-end, sem o uso de frameworks ou JavaScript.
Blog TechDay é um projeto fictício criado com foco em praticar e demonstrar habilidades em HTML e CSS, com estrutura moderna, responsiva e voltada para o portfólio de um desenvolvedor front-end em formação.
Funcionalidades implementadas:
Layout responsivo para desktop, tablet e celular
Cabeçalho com navegação fixa e menu horizontal
Seções temáticas com conteúdo fictício e visual moderno
Tipografia profissional (Google Fonts)
Estilização avançada com flexbox, hover effects, media queries
Estrutura de código limpa e bem comentada.

Tecnologias usadas
HTML5 
CSS3 
Google Fonts (Inter)
Organização em pastas (/img, style.css, index.html)
Planejado para futura expansão com JavaScript
